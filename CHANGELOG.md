## [1.0.0] - 2026-07-14

### Added

- My Craft Server 管理 Web UI（Flask / Gunicorn）
- ダッシュボード（サーバー状態・LAN/外部接続・プレイヤー・ワールド概要）
- サーバー名変更（`/settings`）
- プレイヤー管理（Kick / BAN / 権限 / 削除 / BANリスト）
- ワールド管理（作成・切替・インポート/エクスポート・バックアップ・詳細設定）
- アドオン管理（`.mcpack` / `.mcaddon` / `.zip`）
- 外部接続（Playit.gg / ポート開放）
- システム診断・出荷前テスト
- アップデート・バックアップ
- Discord 通知・リモートサポート
- リセットセンター
- 日本語 / 英語 UI（設定画面）
